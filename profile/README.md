## 📖 组织简介

**异猫工作群（Mutantcat Working Group）** 是一个充满创造力的开源技术团队，专注于开发高质量的**软件工具、游戏与服务**。我们坚持「因为创造，所以存在」的开源信念，欢迎任何人复刻、二次开发并贡献代码。
